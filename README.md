# PNG-converter
Convert PNG to PPM/BMP/ICO
```bash
usage: ./image_converter --input <input.png> --output <output> --format <ppm/bmp/ico>
arguments:
--input, -i   : Input(PNG)
--output, -o  : output(Any name)
--format, -f  : Format (ppm, bmp, ico)
 --help, -h    : Help
```

Compile:
g++ main.cpp
